# 🧪 API Gateway Lab with Flask

This is a simple hands-on lab to demonstrate how an API Gateway works in a microservices architecture using Flask.

## 🛠 Microservices Included
- Product Service (`5001`)
- Order Service (`5002`)
- User Service (`5003`)
- API Gateway (`5000`)
